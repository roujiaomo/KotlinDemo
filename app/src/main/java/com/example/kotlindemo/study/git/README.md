# Git 常用

git clone 将远端的所有分支的镜像和指针clone到本地

git fetch 更新远端clone到本地镜像的位置

git pull 更新远端clone到本地镜像的位置，并将本地分支merge到本地镜像的位置 ：
git pull origin master = git fetch + git merge origin/master (origin/master为远端clone到本地的镜像)
